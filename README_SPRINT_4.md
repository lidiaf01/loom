# Sprint 4 - Historias de Usuario e Implementaciones

## Autenticación y Usuarios
- Gestión de perfil: nombre, email, biografía, foto, visibilidad
- Cambiar contraseña
- Eliminación de cuenta y datos relacionados
- Gestión de seguidores, solicitudes y lista de seguidores/seguidos

## Interfaz y Experiencia de Usuario
- Interfaz responsive y mobile-first (TailwindCSS)
- Iconografía SVG personalizada
- Sistema de toast y modal para feedback y confirmaciones
- Accesibilidad básica (roles, contraste, navegación teclado)
- Diseño moderno y adaptable

## Seguridad y Validación
- Validación de formularios en backend y frontend
- Protección CSRF y XSS
- Hash de contraseñas

## Datos y Almacenamiento
- Migraciones y seeders para todas las entidades
- Integración almacenamiento de archivos (Laravel Storage)
- Relaciones avanzadas en base de datos

## Pruebas y Calidad
- Pruebas automáticas (PHPUnit) y manuales
- Cobertura de casos de uso principales

## API y Externo
- API REST básica para carpetas (Laravel Sanctum)
- Endpoints protegidos y autenticados

## Documentación y Despliegue
- Documentación técnica y de usuario
- Instrucciones de despliegue y uso
- Licencia y publicación
