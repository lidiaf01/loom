# Instrucciones para Iconos - Proyecto Loom

## 📍 Ubicación de Archivos

### Iconos Decorativos
**Carpeta:** `src/www/assets/iconos/decorativos/`

**Formato:** PNG con transparencia
**Tamaños recomendados:**
- Pequeños: 24x24px a 48x48px (para elementos pequeños)
- Medianos: 64x64px a 96x96px (para elementos medianos)
- Grandes: 128x128px a 256x256px (para elementos grandes de fondo)

**Nombres sugeridos:**
- `estrella.png`
- `nube.png`
- `corazon.png`
- `flor.png`
- `luna.png`
- `sol.png`
- etc.

**Uso:** Se distribuirán por toda la aplicación como elementos decorativos de fondo con opacidad baja (0.2-0.3).

---

### Iconos de Botones
**Carpeta:** `src/www/assets/iconos/botones/`

**Formato:** PNG con transparencia
**Tamaño:** 24x24px a 32x32px (todos del mismo tamaño para consistencia)

**Nombres requeridos:**
- `registro.png` - Para botón de registro
- `login.png` - Para botón de login/iniciar sesión
- `crear.png` - Para botón crear
- `buscar.png` - Para botón buscar
- `perfil.png` - Para botón perfil
- `diario.png` - Para botón diario
- `biblioteca.png` - Para botón biblioteca
- `ajustes.png` - Para botón ajustes
- `cerrar-sesion.png` - Para botón cerrar sesión

**Uso:** Se integrarán dentro de los botones junto al texto para hacerlos más visuales.

## 📝 Notas

- Todos los iconos deben tener fondo transparente
- Mantén un estilo consistente entre todos los iconos
- Los iconos decorativos pueden tener diferentes tamaños según su uso
- Los iconos de botones deben ser del mismo tamaño para consistencia visual
- Una vez colocados los archivos, se integrarán automáticamente en la aplicación
