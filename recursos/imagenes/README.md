# Carpeta de Imágenes

Esta carpeta contiene las imágenes del proyecto Loom.

## Avatar por defecto

Para que la aplicación funcione correctamente, necesitas añadir un archivo `default_avatar.png` en esta carpeta.

Puedes:
1. Usar cualquier imagen PNG como avatar por defecto
2. Crear una imagen simple de 100x100px
3. Usar un generador de avatares online

El archivo debe llamarse exactamente: `default_avatar.png`
