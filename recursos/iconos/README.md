# Iconos - Proyecto Loom

## Estructura de Carpetas

```
iconos/
├── decorativos/     ← Iconos decorativos (estrellas, nubes, etc.)
└── botones/         ← Iconos para botones
```

---

## Iconos Decorativos

**Ubicación:** `iconos/decorativos/`

**Formato:** PNG con transparencia
**Tamaño recomendado:** 
- Pequeños: 24x24px a 48x48px
- Medianos: 64x64px a 96x96px
- Grandes: 128x128px a 256x256px

**Nombres sugeridos:**
- `estrella.png`
- `nube.png`
- `corazon.png`
- `flor.png`
- etc.

**Uso:** Se distribuirán por toda la aplicación como elementos decorativos de fondo.

---

## Iconos de Botones

**Ubicación:** `iconos/botones/`

**Formato:** PNG con transparencia
**Tamaño recomendado:** 24x24px a 32x32px

**Nombres sugeridos según función:**
- `registro.png` - Para botón de registro
- `login.png` - Para botón de login
- `crear.png` - Para botón crear
- `buscar.png` - Para botón buscar
- `perfil.png` - Para botón perfil
- `diario.png` - Para botón diario
- `biblioteca.png` - Para botón biblioteca
- `ajustes.png` - Para botón ajustes
- `cerrar-sesion.png` - Para botón cerrar sesión

**Uso:** Se integrarán en los botones de la aplicación para hacerlos más visuales.

---

## Notas

- Todos los iconos deben tener fondo transparente
- Mantén un estilo consistente entre todos los iconos
- Los iconos decorativos pueden tener diferentes tamaños según su uso
- Los iconos de botones deben ser del mismo tamaño para consistencia
