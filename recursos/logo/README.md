# Logo de Loom

Coloca aquí los archivos del logo de Loom.

## Archivos necesarios:

1. **loom-logo.png** - Logo principal (horizontal)
   - Tamaño recomendado: Mínimo 400px de ancho
   - Formato: PNG con transparencia
   - Uso: Header de la aplicación

2. **loom-icon.png** - Icono pequeño (opcional)
   - Tamaño: 32x32 o 64x64px
   - Formato: PNG
   - Uso: Favicon del navegador

## Estructura esperada:

```
logo/
├── loom-logo.png      ← REQUERIDO
└── loom-icon.png      ← OPCIONAL
```
