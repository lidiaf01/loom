# Sprint 1 - Historias de Usuario Completadas

Este documento contiene únicamente las historias de usuario completadas en el Sprint 1, resumidas de forma breve.

### HU-02: Inicio de Sesión (Prioridad: Alta)
- Como: usuario registrado
- Quiero: iniciar sesión con mis credenciales
- Criterios (resumen): formulario email+clave, validación de credenciales, sesión PHP segura, redirección a pantalla principal, opción "Recordar sesión" (cookie)
- Story Points: 3
- Estado: Completado

### HU-03: Pantalla Principal (Prioridad: Media)
- Como: usuario autenticado
- Quiero: acceder a la pantalla principal con navegación
- Criterios (resumen): diseño responsive, menú de navegación (Crear, Búsqueda, Perfil, Diario, Biblioteca, Ajustes), logo visible, protección de ruta (solo usuarios autenticados)
- Story Points: 5
- Estado: Completado
